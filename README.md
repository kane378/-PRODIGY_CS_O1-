# Prodigy InfoTech - Cybersecurity Internship
## Task 01: Caesar Cipher Tool

This repository contains a Python-based tool designed to perform encryption and decryption using the **Caesar Cipher** algorithm. 

### 🛠️ Project Description
The Caesar Cipher is a substitution cipher where each letter in the plaintext is replaced by a letter a certain number of positions down the alphabet. This script allows users to input custom messages and shift values.

### 🚀 Features
* **Encrypts** messages by shifting letters forward.
* **Decrypts** messages by shifting letters backward.
* **Preserves Formatting**: Keeps spaces, numbers, and symbols exactly as they are.
* **Case Sensitive**: Maintains uppercase and lowercase distinctions.

### 📂 File Structure
* `main.py`: The core Python script containing the cipher logic.
* `README.md`: Project documentation.
